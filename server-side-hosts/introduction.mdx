## Overview

Hardal provides built-in server-side hosts that allow you to process and route your analytics data through your own infrastructure. Currently supported hosts:

- [Google Tag Manager Server](/server-side-hosts/gtm-server) - Process and route data through GTM server-side
- [Meta CAPI Gateway](/server-hosts/meta-capi) - Direct server-side conversion tracking

## Benefits of Server-side Hosts

- Improved data quality and reliability
- Better control over data flow
- Reduced client-side code
- Enhanced privacy compliance
- Faster page load times
